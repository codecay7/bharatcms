# 🇮🇳 BharatCMS

> India's first AI-powered headless CMS with built-in Razorpay, GST, Hindi & WhatsApp support.

## Features
- 💳 Razorpay & UPI payments built-in
- 📄 Auto GST-compliant invoicing
- 🤖 AI content generator (Groq)
- 🇮🇳 Hindi & multi-language support
- 💬 WhatsApp notifications
- 🎨 5 ready-to-use templates

## Tech Stack
- Strapi v4.25.x (TypeScript)
- PostgreSQL (Supabase)
- Cloudflare R2 (Storage)
- Razorpay (Payments)
- Groq (AI)
- Resend (Email)

## Quick Start

\`\`\`bash
# Install dependencies
npm install

# Setup environment
cp .env.example .env
# Edit .env with your credentials

# Start development
npm run develop
\`\`\`

Visit `http://localhost:1337/admin` to access the admin panel.
